# Lounge Digital Menu System

A web-based menu platform designed for lounges and bars. Customers access the menu via QR code, while lounge staff manage menu items, prices, and availability through an admin dashboard.

## Features

- **QR Code Access**: Customers scan QR code to access the digital menu
- **Admin Dashboard**: Staff manage menu items, prices, and availability without technical knowledge
- **Automated Temporary Pricing**: Special prices for show days, happy hours, and events that automatically activate and deactivate based on predefined dates and times
- **Events Module**: Create events (DJ night, live band, etc.) that automatically apply temporary pricing during event times
- **Mobile-First Design**: Optimized for mobile devices with fast loading times
- **Menu Categories**: Food, Drinks, and Smoke sections with multiple sub-categories
